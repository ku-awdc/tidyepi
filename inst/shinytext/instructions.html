Instructions

Step 1: Click on the 'Upload' tab and upload one or more Excel/CSV file(s) specifying the metadata, key and one or more datasets

Step 2: Click on the 'Process' button to generate output

Step 3: Click on the 'Results' tab to see the results of the data checking: this will either be a message telling you that the data passed inspection, or details of the errors detected in your data. If there are errors you can either enter corrected values directly into the website (recommended) or adjust the values in your raw data and re-upload.

Step 4: Click on the 'Output' tab

Step 5: Click to download the PDF report file with (graphical) summaries of your uploaded data

Step 6: Click to download the cleaned rdata file, R code to reproduce the import process (including any corrected values from step 3), and R file to reproduce the summary statistic output to your computer

Step 7: Open the rdata and code files in R(Studio) and/or send the rdata file to your friendly neighbourhood epidemiologist

For more information and some examples see [a website that has not yet been created]

If you are using this data checker extensively on shinyapps.io then please consider downloading the software and running it locally: first install R, then type install.packages('devtools') and wait for the output to finish, then type devtools::install_github('ku-awdc/tidyepi') and wait, and finally type tidyepi::tidyepi()
