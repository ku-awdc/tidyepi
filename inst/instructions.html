Instructions

Step 1: Click on the 'Dictionary' tab and upload an Excel/CSV file in the required format

Step 2: Click on the 'Validate' button to ensure that your data dictionary conforms to the requirements

Step 3: Click on the 'Data' tab to upload either a single Excel file or multiple CSV files

Step 4: Click on the 'Check' button to check, clean and tidy your data

Step 5: Click on the 'Output' tab to see the results of the data checking: this will either be a list of the errors detected in your data, or a summary output from the cleaned dataset

Step 6: Download the rdata file and code inspiration file to your computer

Step 7: Open the rdata and code files in R(Studio) and/or send the rdata file to your friendly neighbourhood epidemiologist

For more information and some examples see [a website that has not yet been created]

If you are using this data checker extensively on shinyapps.io then please consider downloading the software and running it locally: first install R, then type install.packages('devtools') and wait for the output to finish, then type devtools::install_github('ku-awdc/tidyepi') and wait, and finally type tidyepi::tidyepi()
